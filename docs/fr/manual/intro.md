# Introduction
TraceMQ est l'outil indispensable pour les développeurs et ingénieurs travaillant avec des protocoles de messagerie tels que MQTT, NATS, Kafka, etc. Il combine un client performant, un outil de simulation avancé et un environnement de scripting pour tester et valider vos systèmes IoT et distribués.

## Installation

### Prérequis
- Windows 10/11 ou Linux.
- Connexion internet (pour les mises à jour).

### Téléchargement
Téléchargez la dernière version depuis la page [Releases](https://github.com/Secureflow-SRL/TraceMQ/releases).

## Premier Démarrage
À l'ouverture, vous accédez au tableau de bord principal. La barre latérale vous permet de naviguer entre les différentes vues :
- **Messages** : Liste en temps réel.
- **Tableau de Bord** : Widgets personnalisables.
- **Topologie** : Carte des topics.
- **Scripts** : Éditeur de scripts.

## Configuration Rapide
1.  Cliquez sur le bouton **"+"** dans la barre latérale pour ajouter une connexion.
2.  Choisissez le protocole (ex: MQTT).
3.  Entrez l'URL du broker (ex: `tcp://localhost:1883`).
4.  Sauvegardez.
5.  Cliquez sur le profil pour vous connecter.
