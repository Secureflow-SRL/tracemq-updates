## Installatie

### Vereisten
- Windows 10/11 of Linux.
- Internetverbinding (voor updates).

### Downloaden
Download de nieuwste versie van de [Releases](https://github.com/Secureflow-SRL/TraceMQ/releases) pagina.

## Eerste Start
Bij het openen komt u op het hoofddashboard. Via de zijbalk kunt u navigeren tussen de verschillende weergaven:
- **Berichten**: Real-time lijst.
- **Dashboard**: Aanpasbare widgets.
- **Topologie**: Topic kaart.
- **Scripts**: Script editor.

## Snelle Configuratie
1. Klik op de **"+"** knop in de zijbalk om een verbinding toe te voegen.
2. Kies het protocol (bijv. MQTT).
3. Voer de broker URL in (bijv. `tcp://localhost:1883`).
4. Sla op.
5. Klik op het profiel om verbinding te maken.
