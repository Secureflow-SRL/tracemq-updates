# Introduction
TraceMQ is the essential tool for developers and engineers working with messaging protocols such as MQTT, NATS, Kafka, etc. It combines a high-performance client, an advanced simulation tool, and a scripting environment to test and validate your IoT and distributed systems.

## Installation

### Prerequisites
- Windows 10/11 or Linux.
- Internet connection (for updates).

### Download
Download the latest version from the [Releases](https://github.com/Secureflow-SRL/TraceMQ/releases) page.

## First Start
Upon opening, you land on the main dashboard. The sidebar allows you to navigate between different views:
- **Messages**: Real-time list.
- **Dashboard**: Customizable widgets.
- **Topology**: Topic map.
- **Scripts**: Script editor.

## Quick Setup
1. Click the **"+"** button in the sidebar to add a connection.
2. Choose the protocol (e.g., MQTT).
3. Enter the broker URL (e.g., `tcp://localhost:1883`).
4. Save.
5. Click on the profile to connect.
